MT4 for Mac and Linux

The increasing popularity of Mac OS X has increased demand for MT4 for Macs. Metaquotes does not provide native for Mac Trader, Manager, or Admin applications, so most Mac users are stuck using a webtrader, having to reboot into Windows, or using a virtual machine, all inconvenient options for Mac users accustomed to oversimplified workflows.

Kenmore Design has developed a way to run native MT4 Trader, Manager, and Admin applications on Mac OS X and Linux without expensive virtual machines, confusing webtraders, or annoying reboots

Mac and Linux MT4 comes whitelabeled with your name and logo just like the Metaquotes version for PC, and the installation procedure is simple, exactly like millions of other native applications for Mac and Linux. Optimized MT4 applications receive the updates that are pushed by Metaquotes and look and work exactly like the native Windows version.

MT4 Application core is not modified during porting and therefore it abides by MetaQuotes rules and terms.
